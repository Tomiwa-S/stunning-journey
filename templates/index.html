Quick overview

This page is a lightweight admin mockup illustrating the TESA table and sample academic records. It's single-file HTML + CSS and includes copyable SQL snippets for quick use in SSMS.

Sample SQL can be pasted into SSMS to create the TESA table and insert demo rows.

-- Create table
CREATE TABLE TESA (
  TESA_FACULTY VARCHAR(150),
  FACULTY_HEAD VARCHAR(150)
);
 
-- Insert sample rows

Connection details

Use these instructions in SSMS to connect to your RDS endpoint.

  • Server name: <your-rds-endpoint>,1433
  • Authentication: SQL Server Authentication
  • Username: <rds-username>
  • Password: <your-password>

Security group

Allow inbound TCP 1433 from your IP. Outbound may remain unrestricted to allow responses.

Notes

Database is in a public subnet for direct SSMS access — ensure IP whitelist and monitoring.
